Movie Search Application
Created By Amrit Stephen Anand

About
------

The Movie Search Application is a web app that allows users to search for desired movies and view details about them.
The project works with the OMDB API and handles JSON data, updating the DOM in response to user input.
 

Functionality
------------------

- Search Function - Allows Users To Search For Movies
- Movie List Display - Displays a list of available movies along with their posters if available
- Movie Details - Displays movie details upon selection
- Responsive Design- The display adapts to accomodate various screen sizes
- API Integration - The app utilizes the OMDB API to fetch information
- Pagination - Pagination is included to display more search results
- Loading Spinner - A loading spinner is displayed while content is being fetched
- Favourites - Favourites are saved in local storage and may be loaded when required

Instructions
-------------

- Enter a valid OMDB API key into the required field. A key may be obtained from the official site, https://www.omdbapi.com/
- Enter the title of a movie to be located and click the search button
- A list of movies are displayed upon searching
- Select a movie to display additional information related to the movie
- Click the star beside a movie's poster to mark it as a favourite

- Click the "load favourites" option to display saved favourites
